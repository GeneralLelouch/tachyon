dependancies
############

The build script expects a few things:
*Visual Studio 2015 (Community is perfectly fine): https://www.visualstudio.com/en-us/downloads/download-visual-studio-vs.aspx
*MSBuild tool: https://www.microsoft.com/en-us/download/details.aspx?id=4815
*CEF Browser plugin - should be in c:\beam\cef_binary : https://bitbucket.org/chromiumembedded/cef
*additional dependancies including vp8, opus, ffmpeg, curl, (and a few others) to be installed C:\beam\dependencies2015 : http://code.fosshub.com/OBS/downloads
*cmake
*QT 5.6
*HM NIS (build script)

building:
#########
git clone https://github.com/WatchBeam/ftl-studio.git
cd ftl-studio
git checkout ftl-ffmpeg
cd build_scripts
build_tachyon_windows_vs.bat
*once complete everything should be located in ftl-studio\build_scripts\rundir\Release

Installer
#########
*after compleleting the above step run:
SET PATH=%PATH%;c:\Program Files (x86)\NSIS
makensis tachyon_windows_installer.nsi

Tachyon_Installer_1.0.exe will be placed in the same folder
